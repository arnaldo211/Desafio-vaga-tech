# Desafio de Projeto DIO: Estruturação de Vagas com IA

## 📒 Descrição
Este repositório contém a resolução do segundo desafio do bootcamp da DIO. O objetivo foi utilizar um prompt estruturado para simular uma entrevista técnica com uma IA, visando definir os requisitos, senioridade, stack tecnológica e soft skills de uma vaga em tecnologia.

## 🤖 Prompt Utilizado
O prompt foi desenhado para atuar como um Entrevistador Técnico, garantindo que todas as dimensões da vaga fossem cobertas de forma organizada (uma pergunta por vez).

**[Link para o arquivo do Prompt](./prompt-entrevistador-vaga.md)**

### 💬 Transcrição da Entrevista

### 📋 Resumo Analítico da Vaga
Após a coleta de dados, a IA gerou o seguinte perfil para a vaga:

- **Título:** Analista de Cibersegurança Júnior (Blue Team)
- **Senioridade:** Júnior (Foco em aprendizado e execução técnica).
- **Stack Principal:** OpenVAS, DefectDojo, SIEM, Firewalls e MITRE ATT&CK.
- **Soft Skills:** Proatividade, atenção aos detalhes e paixão por desafios.

---

## 🛠️ Ferramentas Utilizadas
- **IA Generativa:** Manus AI (utilizando o modelo de Entrevistador Técnico)
- **Controle de Versão:** Git e GitHub

## ✨ Conclusão
Este exercício demonstrou como a IA pode auxiliar gestores e recrutadores a estruturar vagas de forma mais assertiva, garantindo que requisitos técnicos e comportamentais estejam alinhados com a necessidade da equipe.
